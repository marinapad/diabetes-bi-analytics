# Projeto de BI e Analytics - Diabetes

## * Objetivo
Desenvolver uma solução de Business Intelligence e Analytics para identificar fatores associados ao diabetes e prever o risco da doença com base em dados de saúde.



## * Dataset
Base utilizada: Diabetes Health Indicators (BRFSS - CDC)



## * Etapas do Projeto

 1. ETL e Preparação de Dados
- Carregamento do dataset
- Verificação de valores nulos
- Padronização dos dados

 2. Análise Exploratória
- Correlação entre variáveis
- Identificação dos principais fatores de risco

3. Modelagem Preditiva
- Modelo: Random Forest
- Acurácia: 73,83%
- Recall (classe positiva): 78%

 4. BI (Dashboard)
- Indicadores de prevalência
- Análise por idade, IMC e pressão arterial
- Fatores comportamentais e socioeconômicos

---

## * Principais Insights

- IMC é o principal fator de risco
- Idade e condição geral de saúde são altamente relevantes
- Renda e escolaridade impactam o risco
- O modelo apresenta boa capacidade de identificar indivíduos com diabetes
